# Authenticate with the Server

The quickest way to start utilizing the server is to visit the [server](https://llmx.v1.gexapi.com/)

![LLMX UI](/images/llmx-ui.png)

Upon completing the authentication process, it will redirect you to the UI page, which will include your 
API Key. You can use this API Key to make requests to the server.

**Important**: If you run into problems with the authentication process, make sure you clear your cookies and retry.

![Clearing Cookies](/images/clear-cookies.png)
